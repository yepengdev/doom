* 这是什么

一个为 =中英双语写作=、=笔记管理= 和 =学术排版= 深度定制的 Doom Emacs 配置。
运行于 Gentoo Linux，Emacs 30.2，Doom 3.0。

本配置不是面面俱到的 "Emacs 发行版"。它聚焦三个核心场景：

- *写作*：从草稿到排版的完整链路（Olivetti 专注模式 → Org → XeLaTeX + CJK）
- *笔记*：Denote 全家族驱动的知识管理（core + journal + sequence + explore + menu + org）
- *阅读*：PDF（pdf-tools）+ EPUB（nov.el）+ 批注（org-noter）的无缝切换

其他能力（代码编辑、Shell、Git）都围绕这三个核心场景服务。

| 项目 | 值 |
|------+-----|
| 仓库 | ~https://github.com/yepengdev/doom~ |
| 字体 | Maple Mono NF（等宽）· LXGW WenKai（比例）· LXGW WenKai Mono Screen（CJK 回退） |
| 补全 | Corfu + Orderless（行内）/ Vertico（搜索） |
| 国际化 | fcitx5 + evil-pinyin + ace-pinyin + 萌典·mapull 离线词典 |
| C 模块 | count-cjk（字数统计）· cnotify（桌面通知）· random（真随机）· jieba（中文分词） |

* 快速开始

** 日常操作（几乎每天用）

| 命令 | 说明 |
|------+------|
| ~doom sync~ | 修改 ~init.el~ 或 ~packages.el~ 后必须执行 |
| ~M-x doom/reload~ | 修改 ~config.el~ 后热重载 |
| ~M-x my/doom-full-reload~ | 完整重载（autoloads + 包 + config + 主题 + 字体） |
| ~doom upgrade~ | 升级 Doom Emacs 框架本身 |

** 故障排查

| 命令 | 何时使用 |
|------+----------|
| ~doom doctor~ | 行为异常时先跑这个，检查依赖和配置问题 |
| ~doom clean~ | 包损坏或编译缓存问题时，清除后重新 ~doom sync~ |
| ~doom build~ | 重建所有 Doom 包 |
| ~doom env~ | GUI Emacs 找不到环境变量时，重新生成 env 文件 |
| ~make -C c-modules/~ | C 动态模块编译失败时手动重编 |

* 自定义包

本配置通过 ~packages.el~ 引入以下第三方包（均未包含于标准 Doom 模块）：

|----+------+----------|
| ~fcitx~ | 中文输入法与 Evil 状态自动切换 | cjk |
| ~olivetti~ | 写作专注模式（居中 + 隐藏模式行） | 写作工作流 |
| ~palimpsest~ | 搬移文本而非删除（写作修订辅助） | 写作工作流 |

* 三个核心工作流

** 写作
1. ~SPC o o~ 打开 Org 文件，自动进入 ~olivetti-mode~（居中 80 字符宽度）
2. ~super-save~ 在空闲 1 秒后自动保存
3. ~palimpsest~ 确保删除文本进入 kill-ring 可恢复
4. 通过 Org Export（~SPC m e~）导出为 LaTeX → XeLaTeX 编译 PDF

详见 [[file:docs/wiki-index.org][Wiki · workflows/]]。

** 笔记
1. ~SPC n j~ 创建 Denote 日记条目（自动生成标准文件名 + 标题）
2. ~SPC n n~ 创建或跳转到已有笔记
3. ~SPC n s~ 通过 ~consult-notes~ 模糊搜索笔记
4. ~deft~ 提供快速浏览界面（~SPC f d~）
5. ~jieba~ 模块为笔记自动提取中文关键词

详见 [[file:docs/wiki-index.org][Wiki · notes/]]（11 篇深度文档）。

** 阅读
1. ~SPC . p~ 打开 PDF（pdf-tools 增强：搜索、批注、缩放）
2. ~SPC . n~ 打开 EPUB（nov.el 渲染）
3. ~M-x org-noter~ 在 PDF/EPUB 旁创建 Org 批注缓冲区
4. Org 批注通过 ~org-pdftools~ 直接链接回 PDF 对应位置

详见 [[file:docs/wiki-index.org][Wiki · reading/]]。
