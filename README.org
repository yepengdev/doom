* Notable configurations

Set declaratively inside ~(after! doom-ui ...)~:

- ~doom-font~ — Monaspace Neon 16

- ~set-fontset-font~ for CJK — LXGW WenKai Mono Screen 16

** Theme

checked on frame focus (~doom-switch-frame-hook~).

** Org mode
- TODO keywords extended: ~DRAFT~ (R) → ~REVIEW~ (r) → ~CANCELLED~ (C)
- Capture template for novel-writing ideas (~SPC X x~):
  - File: ~~/org/novel-inbox.org~ under "灵感收件箱" heading
  - Fields: title, source, character, emotional tone, notes
- ~org-capture-goto-target~ overridden (adds ~(require 'org-capture)~ for reliable autoloading)
- ~SPC n N~ — go to capture target location
- ~org-directory~: ~~/org/~

** LaTeX (ox-latex via ~config/latex.el~)

Org → LaTeX / PDF publishing configured for Chinese typesetting.

*** Build system

Uses ~latexmk~ instead of manual xelatex multi-pass:

- Compiler: ~latexmk -xelatex -shell-escape -interaction=nonstopmode -f~
- Handles bibtex/makeindex/multiple xelatex passes automatically
- Org's ~org-latex-logfiles-extensions~ cleans auxiliary files on each export

*** Default class: ~ctexbook~

Custom ~ctexbook~ template tailored for Chinese books/notes:

- **No section numbering** (~secnumdepth{0}~), preserves PDF bookmarks
- XeLaTeX-native (no ~inputenc~/~fontenc~, ~hyperref~ loaded manually after ~xcolor~)
- ~minted~ + ~fvextra~ for code listings (breaklines, footnotesize, line numbers)
- ~fancyhdr~ headers/footers, ~geometry~ margins, ~booktabs~ tables
- ~microtype~ (protrusion only), ~caption~, ~enumitem~ (nosep)
- ~\linespread{1.3}~ for comfortable CJK line spacing

*** Custom headline formatting

~org-latex-format-headline-function~ override in ~config/latex.el~:

- TODO keywords rendered in red/green bold sans-serif via ~\texorpdfstring~
- Priority cookies (~[#A]~) shown in orange
- Tags pushed to the right margin in gray
- PDF bookmarks remain clean (raw LaTeX stripped by ~\texorpdfstring~)

*** LaTeX module flags

~init.el~ enables ~(latex +cdlatex +fold)~:

- ~+cdlatex~ — quick LaTeX math/tab snippets (e.g. ~=>~ → ~\Rightarrow~)
- ~+fold~ — TeX environment folding (hide ~\begin~...~\end~ blocks)

*** Removed default packages

- ~inputenc~ — conflicts with XeLaTeX (native UTF-8)
- ~fontenc~ — not needed with OpenType fonts
- ~hyperref~ — loaded manually after ~xcolor~ in template (precise placement)

** Pandoc docx export
- Reference docx template & Lua filters in ~pandoc/~ (version-controlled)
- Default reference: ~template_标题不编号-列表第二行顶格.docx~
- Custom Lua filter: ~markdown-to-docx.lua~ (processed before pandoc's native writer)

** PDF (pdf-tools)
- Default: fit-page display, roll (continuous) scrolling, Evil emacs state
- ~pdf-annot-activate-created-annotations~ — auto-activate new annotations
- Imagemagick scaling disabled

** EPUB (nov.el)
- Auto-major-mode for ~.epub~ files (~nov-mode~)
- Variable-pitch font, visual-line-mode, and olivetti-mode auto-enabled
- hl-line-mode disabled (distracting for prose)
- Reading progress saved to Doom cache dir

** Writing (olivetti)
- Auto-enabled in all Org buffers and nov EPUB buffers
- 100-char width (~olivetti-body-width~)
- Hides mode-line (~olivetti-hide-mode-line~ t) and line numbers
- Toggle manually with ~M-x olivetti-mode~

** Super-save
- Auto-save on idle (~super-save-auto-save-when-idle~)
- Trim trailing whitespace on save (except current line)
- No auto-save on buffer/window switch

** Ace-pinyin
- Global minor mode enabled, integrated with Avy
- Jump to Chinese characters by pinyin initials with ~avy-goto-char-timer~

** Spacious-padding
- Adds padding and breathing room around Emacs frames
- ~line-spacing~: 3
- Daemon-safe: enabled once on first graphical frame (~doom-switch-frame-hook~ with flag)

** Large Org file handling
- Threshold: 1 MiB
- Disables ~org-modern~, ~org-appear~, ~org-indent~, prettify symbols, variable-pitch, and all fontification for files exceeding threshold
- Registered via ~find-file-hook~; non-Org files skipped (short-circuits before ~file-attributes~)

** Startup optimisation
- ~vc-handled-backends~ limited to ~'(Git)~ to skip unnecessary VCS backend stats on file open
- ~auto-mode-case-fold~ set to ~nil~ to avoid per-mode ~file-exists-p~ probing
- ~native-comp-async-jobs-number~ set to 4 with ~native-comp-speed~ 3 for parallel .eln compilation
- ~gcmh~ manages GC dynamically to avoid startup jank from aggressive GC
- ~recentf~ and ~save-place~ entry limits kept modest (100 each)
- Run ~doom compile~ after config changes for byte-compiled load speed

* Gentoo notes

Emacs and some packages (e.g. ~nov~, ~olivetti~, ~org-mode~) are installed
from Portage. After emerging a new ~app-emacs/*~, declare it in ~packages.el~
via ~(package! <name>)~ — Doom picks it up from the site-lisp dir automatically.
