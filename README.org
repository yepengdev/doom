* 已启用的 Doom 模块

源自 ~init.el~，省略注释掉的模块。

** 补全
- ~(corfu +orderless)~ — 行内补全弹窗 + Orderless 模糊过滤
- ~vertico~ — 垂直搜索/选择界面（ivy/helm 的现代替代）

** 界面
- ~deft~ — 快速笔记浏览，从 ~#+title~ 提取标题，清理 Org metadata
- ~doom~ — Doom 主题框架
- ~hl-todo~ — 高亮 ~TODO~ ~FIXME~ ~NOTE~ ~HACK~ ~REVIEW~ ~DEPRECATED~
- ~modeline~ — doom-modeline 模式行
- ~(popup +defaults)~ — 临时缓冲区的弹出规则
- ~(vc-gutter +pretty)~ — 边缘显示版本控制差异指示器（图标版）
- ~workspaces~ — persp-mode 工作区

** 编辑器
- ~(evil +everywhere)~ — 全缓冲区 Vim 按键模拟
- ~file-templates~ — 空文件自动插入模板
- ~fold~ — 通用代码折叠
- ~(format +onsave)~ — 保存时自动格式化
- ~multiple-cursors~ — 多点同时编辑
- ~snippets~ — yasnippet 片段展开
- ~(whitespace +guess +trim)~ — 检测并清理尾部空白

** Emacs 内置
- ~dired~ — 文件管理器（Doom 美化版）
- ~electric~ — 智能缩进
- ~tramp~ — 透明远程文件编辑
- ~undo~ — 持久化撤销树
- ~vc~ — 内置版本控制集成

** 终端
- ~eshell~ — Elisp Shell
- ~vterm~ — 完整 ANSI 终端模拟

** 工具
- ~(eval +overlay)~ — 行内求值结果显示
- ~lookup~ — 文档/定义查找
- ~magit~ — Git 完整前端
- ~pdf~ — PDF 阅读器增强（pdf-tools）

** 语言
- ~emacs-lisp~ — Elisp 开发支持
- ~(latex +cdlatex +fold)~ — LaTeX 编辑（数学/表格快速输入 + 折叠）
- ~markdown~ — Markdown 编辑
- ~(org +journal +noter +pretty +pandoc)~ — Org mode（日记 + 批注 + 美化 + Pandoc 导出）

** 操作系统
- ~tty~ — 终端 Emacs 体验优化

** 配置
- ~(default +bindings +smartparens)~ — 默认键绑定 + 智能括号
