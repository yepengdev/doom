*** ElegantBook 安装与配置

ElegantBook 是适合中英文书籍/笔记排版的 LaTeX 模板，见
[[https://github.com/ElegantLaTeX/ElegantBook]]。

**** 安装方式

1.  **tlmgr（推荐）**

    #+begin_src sh
    sudo tlmgr install elegantbook elegantnote elegantpaper
    #+end_src

2.  **手动安装**（若 CTAN 缺失或需要定制版本）

    #+begin_src sh
    # 下载 .cls / .sty 放到用户 texmf 树
    mkdir -p ~/texmf/tex/latex/elegantbook
    cp elegantbook.cls elegantbook.sty ~/texmf/tex/latex/elegantbook/
    # 刷新 FNDB（= texhash）
    mktexlsr ~/texmf
    #+end_src

    验证：

    #+begin_src sh
    kpsewhich elegantbook.cls
    #+end_src

    输出应为 ~/texmf/tex/latex/elegantbook/elegantbook.cls 或系统路径。

**** 注意事项

- **inputenc 冲突**：xelatex 使用 UTF-8 原生，不再需要 ~inputenc~ 宏包。配置中已从 ~org-latex-default-packages-alist~ 移除。
- **多平台**：配置通过 ~TEXMFHOME=~/texmf~ 支持用户级安装，不影响系统 TeX Live。

**** 排查

| 问题              | 排查命令                                                         |
|-------------------+------------------------------------------------------------------|
| 找不到模板        | ~kpsewhich elegantbook.cls~                                      |
| 搜索路径不对      | ~kpsepath cnf~ 查看 TeX 搜索路径                                 |
| 系统字体看不到    | ~fc-cache -fv~（xelatex）；~luaotfload-tool -fu~（LuaTeX）       |
| 用户级安装不生效  | 检查 ~TEXMFHOME~ 是否指向 ~/texmf~；运行 ~mktexlsr~ 后重试       |

** PDF (pdf-tools)
- Default: fit-page display, roll (continuous) scrolling, Evil emacs state
- ~pdf-annot-activate-created-annotations~ — auto-activate new annotations
- Imagemagick scaling disabled

** EPUB (nov.el)
- Auto-major-mode for ~.epub~ files (~nov-mode~)
- Variable-pitch font, visual-line-mode, and olivetti-mode auto-enabled
- hl-line-mode disabled (distracting for prose)
- Reading progress saved to Doom cache dir

** Writing (olivetti)
- Auto-enabled in all Org buffers and nov EPUB buffers
- 100-char width (~olivetti-body-width~)
- Hides mode-line (~olivetti-hide-mode-line~ t) and line numbers
- Toggle manually with ~M-x olivetti-mode~

** Super-save
- Auto-save on idle (~super-save-auto-save-when-idle~)
- Trim trailing whitespace on save (except current line)
- No auto-save on buffer/window switch

** Ace-pinyin
- Global minor mode enabled, integrated with Avy
- Jump to Chinese characters by pinyin initials with ~avy-goto-char-timer~

** Spacious-padding
- Adds padding and breathing room around Emacs frames
- ~line-spacing~: 3
- Daemon-safe: enabled once on first graphical frame (~doom-switch-frame-hook~ with flag)

** Large Org file handling
- Threshold: 1 MiB
- Disables ~org-modern~, ~org-appear~, ~org-indent~, prettify symbols, variable-pitch, and all fontification for files exceeding threshold
- Registered via ~find-file-hook~; non-Org files skipped (short-circuits before ~file-attributes~)

** Startup optimisation
- Previously deferred ~file-name-handler-alist~ until ~doom-init-ui-hook~ (removed per Doom upstream advice: Doom's own lazy loading handles this more safely; the old trick could interfere with remote files and archive-mode)
- GC tuning deliberately avoided (caused PDF flicker in previous testing)

* Gentoo notes

Emacs and some packages (e.g. ~nov~, ~olivetti~, ~org-mode~) are installed
from Portage. After emerging a new ~app-emacs/*~, declare it in ~packages.el~
via ~(package! <name>)~ — Doom picks it up from the site-lisp dir automatically.
