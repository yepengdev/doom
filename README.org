** Lang

* Notable configurations

** Fonts
~my/apply-fonts~ runs on both ~after-init-hook~ and ~server-after-make-frame-hook~,
guarded by ~(display-graphic-p frame)~ for daemon safety:

- Monaspace Neon (English, 15pt; variable-pitch 16pt)
- LXGW WenKai Mono Screen (Chinese / CJK, 16pt)

Apply with ~M-x my/apply-fonts~ or ~M-x doom/reload-font~.

** Theme
~doom-wilmersdorf~ — a warm, high-contrast dark theme. Set via ~doom-theme~.

** Line numbers
~relative~ style — display relative line numbers for Evil motion efficiency.

** Chinese input method
Fcitx5 integration via ~fcitx~ package:
- Auto-detects ~fcitx5-remote~ binary at startup
- Integrates with Evil: automatically toggles Chinese IM on insert-mode entry
- Loaded lazily via ~run-with-idle-timer~ (0.5s after init)
- Also integrates with Vertico for pinyin-based file search (~evil-pinyin~)

** Org mode
- TODO keywords extended: ~DRAFT~ (R) → ~REVIEW~ (r) → ~CANCELLED~ (C)
- Capture template for novel-writing ideas (~SPC X x~):
  - File: ~~/org/novel-inbox.org~ under "灵感收件箱" heading
  - Fields: title, source, character, emotional tone, notes
- ~org-directory~: ~~/org/~

** LaTeX (ox-latex)
Configured for CJK / Chinese publishing:

- Compiler: ~xelatex~ (3 passes + ~bibtex~)
- Default class: ~elegantbook~ (Chinese template)
- ~inputenc~ removed from default packages (conflicts with xelatex)
- Code highlighting via ~minted~ + ~fvextra~ (breaklines, footnotesize, line numbers)
- Log files auto-cleaned (~.aux~, ~.log~, ~.bbl~, etc.)

** PDF (pdf-tools)
- Default: fit-page display, roll (continuous) scrolling, Evil emacs state
- ~pdf-annot-activate-created-annotations~ — auto-activate new annotations
- Imagemagick scaling disabled

** Writing (olivetti)
- Auto-enabled in all Org buffers
- 100-char width (~olivetti-body-width~)
- Hides mode-line (~olivetti-hide-mode-line~ t) and line numbers
- Toggle manually with ~M-x olivetti-mode~

** Super-save
- Auto-save on idle (~super-save-auto-save-when-idle~)
- Trim trailing whitespace on save (except current line)
- No auto-save on buffer/window switch

** Ace-pinyin
- Global minor mode enabled, integrated with Avy
- Jump to Chinese characters by pinyin initials with ~avy-goto-char-timer~

** Spacious-padding
- Adds padding and breathing room around Emacs frames
- ~line-spacing~: 3
- Daemon-aware: enables after frame creation with idle timer

** Large Org file handling
- Threshold: 1 MiB
- Disables ~org-modern~, ~org-appear~, ~org-indent~, prettify symbols, variable-pitch, and all fontification for files exceeding threshold
- Inactive by default (commented out in config)

* Gentoo notes

Emacs and some packages (e.g. ~nov~, ~olivetti~, ~org-mode~) are installed
from Portage. After emerging a new ~app-emacs/*~, declare it in ~packages.el~
via ~(package! <name>)~ — Doom picks it up from the site-lisp dir automatically.
