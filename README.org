个人 Doom Emacs 配置，针对中英双语写作、笔记管理（Denote）和学术排版
（XeLaTeX + CJK）深度优化。运行于 Gentoo Linux + Emacs 30.2。

* 快速上手

| 命令 | 何时使用 |
|------+----------|
| ~doom sync~ | 编辑 ~init.el~ 或 ~packages.el~ 后必须执行 |
| ~doom doctor~ | 检查配置问题 |
| ~doom upgrade~ | 更新 Doom Emacs 框架 |
| ~doom clean~ | 清除编译缓存 / elpa 目录 |
| ~doom env~ | 为 GUI Emacs 重新生成环境变量文件 |
| ~M-x doom/reload~ | 重新求值 ~config.el~ |
| ~M-x my/doom-full-reload~ | 完整重载：autoloads + 包 + config + 主题 + 字体（~SPC h r R~） |
| ~M-x doom/reload-font~ | 重新应用字体设置 |
| ~doom build~ | 重建 Doom 包 |
| ~make -C c-modules/~ | 手动编译所有 C 模块 |

* 自定义包

通过 ~packages.el~ 声明的第三方包：

| 包 | 用途 |
|----+------|
| ~fcitx~ | 中文输入法与 Evil 状态集成 |
| ~spacious-padding~ | 界面内边距与呼吸感 |
| ~nov~ | EPUB 阅读器 |
| ~org-pdftools~ | Org 链接到 PDF 位置 |
| ~super-save~ | 空闲时自动保存 |
| ~palimpsest~ | 搬移文本而非删除（写作辅助） |
| ~olivetti~ | 写作专注模式（居中 + 隐藏模式行） |
| ~ace-pinyin~ | 基于 Avy 的拼音字符跳转 |
| ~evil-pinyin~ | 基于拼音的模糊搜索（Orderless 集成） |

萌典和 mapull 使用 Emacs 内置 sqlite，无外部包依赖。
