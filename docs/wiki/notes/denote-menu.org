#+TITLE: 表格视图 (denote-menu)

* 用途

~SPC r d m~ 打开，以表格形式（~tabulated-list-mode~）浏览笔记库，
按列排序和筛选。

* 显示

| 列 | 内容 |
|-----+------|
| File | 笔记文件名 |
| Title | 笔记标题 |
| Keywords | 关键词列表 |
| File type | 文件类型（org/md/txt） |
| File signature | 序列签名（如 1.1.a） |
| Directory | 子目录 |

默认按日期降序排列（最新在前）。

* 筛选

| 按键 | 说明 |
|------+------|
| ~c~ | 清除所有筛选 |
| ~/ r~ | 正则表达式筛选（标题+关键词） |
| ~/ k~ | 筛选包含某关键词的笔记 |
| ~/ o~ | 排除某关键词 |

* 批量操作

~e~ 将当前筛选结果导出到 Dired buffer，然后可以：
- 标记多个文件
- ~d~ ~x~ 批量删除
- ~R~ 批量重命名
- ~!~ 批量执行 shell 命令
