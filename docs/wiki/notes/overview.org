* Denote 文件命名规范

#+BEGIN_SRC text
IDENTIFIER--TITLE__KEYWORDS.EXT
#+END_SRC

示例：
~20240322T131856--读书笔记__思考.org~
~20240322T142301--项目启动方案__工作_技术.org~

- ~IDENTIFIER~ = 时间戳（ISO 8601, 精确到秒）, e.g. ~20240322T131856~
- ~TITLE~ = 标题（~--~ 双短横分隔符前）
- ~KEYWORDS~ = 分类关键词（~__~ 双下划线分隔符后，~_~ 单个下划线分隔多个关键词）
- ~EXT~ = ~.org~（默认）或 ~.md~, ~.txt~

* jieba 自动关键词

创建新笔记时，~my/denote-jieba-infer-keywords~（hook 优先级 80）自动运行：
1. 对笔记全文做 jieba TF-IDF 提取 top 5 关键词
2. 与 ~denote-known-keywords~（以及已有关键词）做交集匹配
3. 命中的关键词自动写入文件 front-matter
4. 文件名同步更新

详见 [[./jieba-keywords.org][jieba-keywords.org]]。
