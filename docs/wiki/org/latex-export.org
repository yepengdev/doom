#+TITLE: LaTeX 导出（XeLaTeX + ctexbook）
#+STARTUP: overview

* 引擎选择

#+BEGIN_SRC emacs-lisp
(setq-default TeX-engine 'xetex)
#+END_SRC

XeLaTeX 是唯一能原生处理中文 + OpenType 字体的 LaTeX 引擎。PDFTeX 需要 ~CJKutf8~ 宏包，
且无法直接使用 TTF/OTF 字体。LuaLaTeX 是另一个选项，但对小型文档启动较慢。

* 编译命令

#+BEGIN_SRC emacs-lisp
(after! ox-latex
  (let* ((modules-dir (expand-file-name "modules" doom-user-dir))
         (cmd (format "env TEXINPUTS=%s//: latexmk -xelatex -shell-escape -interaction=nonstopmode -f -output-directory=%%o %%f"
                      modules-dir)))
    (setq org-latex-pdf-process (list cmd)))
#+END_SRC

- ~latexmk~ — 自动处理多次编译（biber/minted 所需）
- ~-xelatex~ — XeLaTeX 引擎
- ~-shell-escape~ — minted 调用 pygments 所需
- ~TEXINPUTS=%s//:~ — 使 LaTeX 在 ~modules/~ 目录中搜索 ~ctexbook-org.cls~
- ~-output-directory=%o~ — 输出到 Org 的 ~:EXPORT_FILE_NAME:~ 指定的目录

* LaTeX 类：ctexbook

自定义类模板，针对中文书籍/笔记排版。定义文件：~modules/ctexbook-org.cls~。

#+BEGIN_SRC emacs-lisp
(after! ox-latex
  (add-to-list 'org-latex-classes
               '("ctexbook"
                 "\\documentclass{ctexbook-org}
[PACKAGES]
[EXTRA]"
                 ("\\chapter{%s}" . "\\chapter*{%s}")
                 ("\\section{%s}" . "\\section*{%s}")
                 ("\\subsection{%s}" . "\\subsection*{%s}")))
  (setq org-latex-default-class "ctexbook"))
#+END_SRC

** 类定义基础

~ctexbook-org.cls~ 继承自 ~ctexbook~ 文档类：

| 项目 | 设定 |
|------+------|
| 引擎 | XeLaTeX + UTF-8 |
| 字体 | Fandol（CTeX 自带中文字体） |
| 纸张 | 11pt, a4paper, twoside |
| 页码深度 | ~secnumdepth{0}~（正文不编号；PDF 书签保留编号） |
| 页面边距 | ~top=2.5cm, bottom=2.5cm, inner=3cm, outer=2cm~ |
| 行距 | ~1.3~ |

** 使用宏包

| 宏包 | 用途 |
|------+------|
| ~xcolor~ | 标题/TODO 关键词着色 |
| ~microtype~ | 微排版（XeLaTeX 下仅 protrusion） |
| ~fancyhdr~ | 页眉页脚（左页章名、右页节名） |
| ~hyperref~ | 彩色超链接（link=blue, cite=teal） |
| ~xurl~ | URL 折行 |
| ~minted~ | 代码高亮（pygments 后端） |
| ~fvextra~ | Verbatim 环境折行 |
| ~booktabs~ | 三线表 |
| ~caption~ | 图表标题格式化 |
| ~enumitem~ | 列表间距（nosep） |

** 标题格式化：~my/org-latex-format-headline~

#+BEGIN_SRC emacs-lisp
(defun my/org-latex-format-headline (todo todo-type priority text tags _info)
  (concat
   (and todo
        (let ((fmt (pcase todo-type
                     ('todo "{\\color{red!65!black}\\bfseries\\sffamily %s}")
                     ('done "{\\color{green!45!black}\\bfseries\\sffamily %s}")
                     (_ "{\\bfseries\\sffamily %s}")))
          (format "\\texorpdfstring{%s }{%s }" (format fmt todo) todo)))
   (and priority
        (let* ((pri-str (org-priority-to-string priority))
               (colored (format "{\\color{orange!60!black}\\small\\sffamily [\\#%s]}" pri-str)))
          (format "\\texorpdfstring{%s }{%s }" colored (format "[\\#%s]" pri-str))))
   text
   (and tags
        (let ((tag-text (mapconcat #'org-latex--protect-text tags ":")))
          (format "\\texorpdfstring{\\hfill{}{\\color{gray!50!black}\\small %s}}{}" tag-text)))))

(after! ox-latex
  (setq org-latex-format-headline-function #'my/org-latex-format-headline))
#+END_SRC

关键技巧：使用 ~\texorpdfstring{}{}~ 包裹所有 LaTeX 命令。
- 第一参数：PDF 正文中显示的格式化文本（含 ~\color~ 等命令）
- 第二参数：PDF 书签中显示的纯文本（无格式）
- 没有这个，书签中会显示 ~{\color{red!65!black}DRAFT}~ 的原始代码

| TODO 类型 | 颜色 | 字体 |
|-----------+------+------|
| ~todo~（DRAFT/REVIEW） | ~red!65!black~ | ~\bfseries\sffamily~ |
| ~done~（DONE） | ~green!45!black~ | ~\bfseries\sffamily~ |
| ~[#A]~ 优先级 | ~orange!60!black~ | ~\small\sffamily~ |
| ~:tag:~ | ~gray!50!black~ | ~\hfill~ 右对齐 |

* 代码高亮后端

#+BEGIN_SRC emacs-lisp
(after! ox-latex
  (setq org-latex-src-block-backend 'minted))
#+END_SRC

使用 minted + pygments 而非 listings。优势：
- 支持更多语言
- 更好的 Unicode 和 CJK 支持
- 更丰富的样式选择

依赖：~pip install pygments~。

* 日志清理

#+BEGIN_SRC emacs-lisp
(after! ox-latex
  (setq org-latex-logfiles-extensions
        '("lof" "lot" "tex~" "aux" "idx" "log" "out" "toc" "nav" "snm"
          "vrb" "dvi" "fdb_latexmk" "blg" "brf" "fls" "entoc" "ps" "spl"
          "bbl" "tex" "bcf")))
#+END_SRC

编译完成后自动删除这些中间文件，保持工作目录干净。

* 移除的默认包

#+BEGIN_SRC emacs-lisp
(org-latex-default-packages-alist
 (cl-remove '("" "hyperref" t)
            (cl-remove '("T1" "fontenc" t)
                       (cl-remove '("AUTO" "inputenc" t)
                                  org-latex-default-packages-alist
                                  :test #'equal)
                       :test #'equal)
            :test #'equal))
#+END_SRC

| 移除的包 | 原因 |
|----------+------|
| ~inputenc~ | XeLaTeX 原生 UTF-8，~inputenc~ 会破坏编码检测 |
| ~fontenc~ | OpenType 字体不需要 T1 编码 |
| ~hyperref~ | 在 ~ctexbook-org.cls~ 中手动加载（放在 color 之后、metalink 之前，顺序重要） |

* 注意事项

- 需要安装 ~minted~ 包：~pip install pygments~
- 需要 Fandol 字体（CTeX 自带，但某些发行版需要单独安装 ~texlive-lang-chinese~）
- ~-shell-escape~ 开启可带来安全风险——只在信任的文档中使用
- LaTeX 类中 ~[DEFAULT-PACKAGES]~ 和 ~[PACKAGES]~ 占位符由 Org 在导出时替换
