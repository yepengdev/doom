| 国际化 | fcitx5 集成、evil-pinyin/ace-pinyin |
| 工具 | 完整重载命令、byte-compile、ispell 配置 |

** Evil Insert 混合模式

- ~evil-insert-state-map~ 清空，仅保留 ~<escape>~ → Normal state
- 其他按键回退到 Emacs 全局键绑定（~C-w~ ~C-a~ ~C-e~ 等工作）
- 光标为竖线（~bar~）
- 保持了 ~evil-insert-state~ 身份不变（第三方包兼容）

** Olivetti 写作专注

- 用途：居中正文、减少视觉宽度、隐藏 mode-line
- ~olivetti-body-width~ = 100，~olivetti-hide-mode-line~ = ~t~
- 自动在 ~org-mode~ 中启用
- Olivetti 启用时关闭行号，退出时恢复

** Super-save 自动保存

- ~doom-first-file~ 时启用，空闲时自动保存
- 静默模式（不显示 minibuffer 消息）
- ~super-save-delete-trailing-whitespace~ = ~'except-current-line~
- 只读 buffer 不触发

** Palimpsest 搬移文本

- 三个目的地：buffer 顶部、buffer 底部、~<basename>.trash.<ext>~
- 绑定在 ~SPC m P~（~t~ / ~b~ / ~T~）
- 写作中替代 ~dd~ / ~x~ 删除，避免 undo 链污染

** Pandoc docx 导出

- 资源目录：~pandoc/~（相对于 ~doom-user-dir~）

** HTML 导出

- 本地 CSS 方案（~org/export/minimal/css/~），零 JS，无 CDN

** LaTeX 导出

- 引擎：XeLaTeX（~TeX-engine~ = ~xetex~）
- 构建：~latexmk -xelatex -shell-escape~
- 文档类：~ctexbook~（自定义，基于 ctexbook）
- 字体：Fandol（CTeX 自带）
- 标题格式化：~my/org-latex-format-headline~（TODO/PRIORITY/TAG 着色，PDF 书签纯文本）

** 大文件处理（≥ 1 MiB）

自动关闭所有装饰：org-modern、org-appear、org-indent、prettify-symbols、variable-pitch、fontification。

** Org 实时预览

~M-x my/org-live-preview~ — 启动当前 buffer 的 HTML 实时预览。
机制：导出 → Python HTTP+SSE 服务器 → 浏览器实时刷新。
依赖：Python 3 + ~scripts/org-live-server.py~。

* 国际化

** 中文输入（fcitx5）

- 通过 ~doom-first-input-hook~ 延迟加载（启动时不启动 fcitx5）
- 自动检测 ~fcitx5-remote~ 或 ~fcitx-remote~
- 配合 Evil Insert 混合模式，减少 IME 切换事件

** 拼音模糊搜索

- ~evil-pinyin~：advice ~orderless-regexp~，拼音匹配中文（如 ~zhongwen~ → ~中文~）
- 守卫机制：仅处理纯 ASCII 字母输入（长度 < 10），避免大字符集正则溢出
- ~ace-pinyin~：Avy 字符跳转支持拼音，~f/F~ ~t/T~ 可跳转中文

** 中文字数统计

使用 count-cjk C 模块（~c-modules/count-cjk.so~），单次 UTF-8 扫描完成：
- ~my/count-chinese-chars~（~SPC r n c~）：汉字数 + 含标点数 + 百分比
- ~my/count-words~（~M-==~）：中/英/标点综合统计
- ~my/word-count~（~SPC r n w~）：详细信息（CJK + punct + EN words + EN chars + total）
- ~my/writing-stats-mode~（~SPC r n s~）：mode line 实时显示写作统计

** 中文词典

~SPC o d~ / ~K~ — 统一入口：
- 单汉字 → 萌典释义 + 汉字档案 + 相关词语/成语
- 多汉字 → mapull 词语/成语
- 非中文 → 原始英语后端

mapull 首次查询自动下载约 100MB 数据并导入 sqlite。
萌典需要自行准备 ~dict.sqlite3~ 到 ~doom-local-dir/dict/~。

* 工具

** 完整重载（~my/doom-full-reload~）

~SPC h r R~。比 ~doom/reload~ 更彻底：
1. ~doom/reload-autoloads~ — 重载 autoloads
2. ~doom/reload-packages~ — 重载包配置
3. ~doom/reload~ — 重载 config.el
4. 自动重应用主题 + 字体（~doom-after-reload-hook~）

** byte-compile

~M-x my/byte-compile-config~ 编译 ~config.el~ + ~cli.el~ + 所有自定义模块 ~config.el~。
当前会话继续用 source，下次启动自动加载 ~.elc~。

** Git 代理

通过 ~DOOMGITCONFIG~ 环境变量为 Doom 内部 Git 操作设置 GitHub 镜像代理（gh-proxy.com）。
不影响系统 Git。

** 在线搜索

~SPC s o~（~+lookup/online~）自定义搜索引擎列表：
Bing、Bilibili、抖音、Emacs China、Emacs Wiki、知乎、术语在线、求闻百科。
通过 ~niri msg~ 自动将焦点切换到浏览器窗口（Wayland + niri 窗口管理器）。

** 番茄钟与计时器

基于 cnotify C 模块（~c-modules/cnotify-module.so~）：
- 模式行倒计时显示（🍅 工作 / ☕ 休息 / ⏱ 计时）
- 完成自动记录到 ~pomodoro.eld~ 日志
- 统计命令查看今日/本周/全部番茄钟数

** 随机密码

~my/random-password~（~SPC r t p~）使用 random C 模块（~c-modules/random.so~）
生成 24 字符真随机密码，自动复制到剪贴板。

** ispell

- ~ispell-dictionary~ = ~"en_US"~（强制英文词典，避免 LC_CTYPE=zh_CN 下字典不兼容）
- 手动从 ~completion-at-point-functions~ 移除 ispell（未启用 spell checker 模块）
