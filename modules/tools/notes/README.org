* Description :unfold:
This module configures a complete note-taking workflow centred around
[[https://protesilaos.com/emacs/denote][Denote]] (file-naming Zettelkasten), with [[doom-package:deft]] for quick file
browsing and [[doom-package:consult-notes]] for search integration.

This module has no external dependencies beyond the Elisp packages listed above.

* Usage
** Deft (file browser)
Deft is configured to search =~/org/deft/= recursively. Org-mode titles
( =#+title:= ) are parsed and used as display names.

** Denote (primary note system)
Notes live in =~/org/denote/=, with journal entries in =~/org/denote/journal/=.

** Auto-commit
A git auto-commit hook is installed on ~denote-after-new-note-hook~: every new
note triggers =git add -A && git commit --allow-empty= in the Denote directory.
The hook checks ~call-process~ return codes and logs warnings on failure.

** jieba keyword auto-tagging
If the [[doom-module::tools jieba]] module is enabled, ~denote-after-new-note-hook~
also runs a TF-IDF keyword extractor (at priority 80). It extracts the top 5
keywords from the new note, matches them against ~denote-known-keywords~, and
auto-renames the file to include matched keywords in its front-matter.

* Configuration
Set the following variables in your =config.el= *before* enabling this module:

#+begin_src emacs-lisp
;; Optional: change Denote directory (default: ~/org/denote)
(setq denote-directory "~/org/denote")
;; Optional: change journal directory
(setq denote-journal-directory "~/org/denote/journal")
#+end_src

* Film→Writing (~+film~ flag)
When the ~+film~ flag is enabled ( =:tools (notes +film)= ), a film-for-writing
workflow is added on top of the Denote core.  Every film viewed is treated as
a source of craft lessons for novel writing.

Film notes live in =~/org/denote/film/= (a Denote silo).

** Keybindings (~SPC r d v~ prefix, "v" for 观影)

| Key | Command                          | Description                      |
|-----+----------------------------------+----------------------------------|
| ~n~ | ~my/denote-new-film~              | New film note (writing template) |
| ~s~ | ~my/denote-find-film~             | Search film notes (grep)         |
| ~i~ | ~my/open-film-index~              | Open film index (org table)      |
| ~a~ | ~my/film-add-to-index~            | Add current note to index        |
| ~c~ | ~my/film-stats~                   | Show film count                  |
| ~m~ | ~my/film-migrate-note~            | Migrate this note to new format  |
| ~M~ | ~my/film-migrate-all~             | Migrate all notes to new format  |
| ~e~ | ~my/film-export-writing-notes~    | Export writing snippets to file  |

** Template sections
New film notes via ~SPC r d v n~ prompt for title (required), director and year
(optional).  The template provides six writing-analysis sections:

- 叙事结构分析
- 人物弧光
- 可借鉴的写作手法
- 场景与氛围
- 对白分析
- 主题与隐喻
- 灵感触发 / 写作素材

** Migration from old format
Run ~SPC r d v m~ (current note) or ~M~ (batch) to upgrade old film diary
notes to the new writing template.  Old sections are mapped as follows:

| Old                         | New                            |
|-----------------------------+--------------------------------|
| 为什么看这部片子            | 灵感触发 / 写作素材            |
| 印象深刻的场景              | 场景与氛围                     |
| 配乐/摄影/表演              | 可借鉴的写作手法（附来源说明）|
| 一句话总结                  | 灵感触发（子标题** 一句话总结）|
| RATING /10                  | RATING（去掉 /10 后缀）        |

** Writing-snippet export
~SPC r d v e~ collects all "可借鉴的写作手法" and "灵感触发 / 写作素材"
sections across every film note and writes them into
=~/org/film-writing-snippets.org=, each entry linked back to the original note
via =denote:=.  Useful as a reference while drafting.

** Org-capture template (~M-x org-capture~ ~f~)
Minimal: title, director, year — enough for a quick log entry in
=~/org/film-diary.org=.  Detailed analysis goes into the full Denote note.

** Known keywords

Default keywords: ~'("创作" "学习" "工作" "生活" "技术" "思考" "索引")~
