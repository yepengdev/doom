#+TITLE: 自定义函数索引

| 函数 | 定义位置 | 用途 |
|------+----------+------|
| ~my/doom-full-reload~ | config.el | 完整重载配置 + 主题 + 字体 + frames |

| ~my/theme-for-hour~ | config.el | 根据小时返回主题符号 |
| ~my/theme-apply~ | config.el | 设置并刷新主题 |
| ~my/theme-switch-maybe~ | config.el | 首次 frame 切换时自卸式检查主题 |
| ~my/enable-spacious-padding--fn~ | config.el | 启用 spacious-padding（daemon-safe） |
| ~my/dired-open-externally~ | config.el | 外部程序打开标记文件 |
| ~my/deft-parse-title~ | config.el | 解析 Org 标题（大小写不敏感） |
| ~my/denote-git-auto-commit~ | config.el | 笔记创建后自动 git commit |
| ~my/org-latex-format-headline~ | config.el | 带颜色 /texorpdfstring 的 LaTeX 标题格式 |
| ~my/org-maybe-disable-prettification~ | config.el | ≥1 MiB Org 文件关闭美化 |
| ~my/count-chinese-chars~ | config.el | 中文字数统计 |
| ~org-capture-goto-target~ | config.el | 跳转到 capture 模板目标位置 |
