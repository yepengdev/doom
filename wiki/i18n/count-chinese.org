#+TITLE: 字数统计 (my/count-words / my/count-chinese-chars)

* 用途

中英双语统一统计。C 动态模块（~c-modules/count-cjk.so~）一次扫描完成所有计数。

| 功能 | 快捷键 | 输出示例 |
|------|--------|----------|
| 综合统计（中/英/标点/总） | ~M-=~ | ~中:12  英:2  标点:2  总:28~ |
| CJK 字数（旧接口） | ~SPC r n c~ | ~字:12  含标点:14  总:28  50.0%~ |

* 架构

C 模块导出两个函数：

- ~my/count-cjk (STRING)~ → ~(CHARS . PUNCT)~（CJK 字和标点）
- ~my/count-text (STRING)~ → ~[cjk punct en-words en-chars total-cp]~（全量统计）

Elisp 层只做 message 格式化，计算由 C 完成。

* 英文词界定

~[a-zA-Z0-9']~ 连续序列为一个英文词：

- ~Hello~ → 1 词
- ~don't~ → 1 词（~'~ 视为词内字符）
- ~well-known~ → 2 词（~-~ 是词边界）
- ~42~ → 1 词（数字参与）

* C 模块构建

C 模块使用惰性加载策略：启动时不编译不加载，首次调用命令时按需编译并加载。
也可手动触发构建：

#+BEGIN_SRC sh
cd ~/.config/doom && make -C c-modules/
#+END_SRC

或 Emacs 内 ~M-x my/build-cjk-module~（~SPC r n b~）。

需 ~emacs-module.h~（Emacs 30 开发头文件）。

* Unicode 覆盖

CJK 统一表意文字及其扩展区块（A–F），合计约 ~97,000 字。UTF-8 解码带边界检查，拒绝 overlong / surrogate / 截断序列。
