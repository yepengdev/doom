#+TITLE: Git 自动提交

* 用途

每次创建新笔记后自动执行 ~git add -A && git commit~，作为安全网——
即使忘记手动提交，笔记也不会丢失。

* 配置

#+BEGIN_SRC emacs-lisp
(defun my/denote-git-auto-commit ()
  (when-let ((dir (denote-directory)))
    (let ((git-dir (expand-file-name ".git" dir)))
      (when (file-exists-p git-dir)
        (let ((default-directory dir))
          (call-process "git" nil nil nil "add" "-A")
          (call-process "git" nil nil nil "commit" "-m" "auto: note saved"))))))
(add-hook 'denote-after-new-note-hook #'my/denote-git-auto-commit)
#+END_SRC

* 设计决策

| 决策 | 理由 |
|------+------|
| 仅在 .git 存在时执行 | 笔记目录可能不是 git 仓库（如首次配置时） |
| ~call-process~ 非阻塞 | 不用 ~shell-command~（避免弹出 *Async Shell Command*） |
| ~nil nil nil~ 参数 | 不等待输出、不显示 buffer、不重定向 stdin |
| commit message ~"auto: note saved"~ | 统一前缀，方便 ~git log --grep="auto:"~ |

* 局限

- 不 handle 冲突（如两个 Emacs 实例同时创建笔记）
- 不 handle 网络（不 push，只是本地 commit）
- 不覆盖编辑已有笔记的保存（由 editor/super-save 处理）
