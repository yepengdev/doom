#+TITLE: Pandoc docx 导出
#+STARTUP: overview

* 用途

将 Org 文件导出为 Microsoft Word (.docx) 格式，用于与不使用 LaTeX/Org 的协作者交换文稿。

* 资源目录

#+BEGIN_SRC emacs-lisp
(defvar my/pandoc-dir (expand-file-name "pandoc" doom-user-dir))
#+END_SRC

所有 Pandoc 资源位于 ~/.config/doom/pandoc/~。

* 默认配置

#+BEGIN_SRC emacs-lisp
(after! ox-pandoc
  (setq org-pandoc-options-for-docx
        `((reference-doc . ,(expand-file-name
                             "templates/template_标题不编号-列表第二行顶格.docx"
                             my/pandoc-dir))
          (lua-filter . ,(expand-file-name "markdown-to-docx.lua" my/pandoc-dir)))))
#+END_SRC

- 参考文档（reference-doc）：docx 模板，pandoc 从中提取样式定义
- Lua filter：pandoc 的 lua 过滤器管线入口

* 模板清单

~pandoc/templates/~ 下 7 个模板：

| 模板 | 特点 | 适用场景 |
|------+------+----------|
| ~template_sci论文-标题不编号~ | 学术论文风格，无编号 | 论文 |
| ~template_sci论文-标题编号~ | 编号 | 论文 |
| ~template_标题不编号-列表第二行缩进~ | 无编号、缩进 | 一般写作 |
| ~template_标题不编号-列表第二行顶格-无首行缩进~ | 无编号、顶格、无首行缩进 | 公文 |
| ~template_标题不编号-列表第二行顶格~ | ★ 默认：无编号、顶格 | 一般写作 |
| ~template_标题编号-列表第二行缩进~ | 编号、缩进 | 技术文档 |
| ~template_标题编号-列表第二行顶格~ | 编号、顶格 | 技术文档 |

* Lua Filter 管线

~markdown-to-docx.lua~ 调用 ~pandoc/lua/~ 下的子过滤器：

| 过滤器 | 作用 |
|--------+------|
| ~add-inline-code.lua~ | 内联代码格式处理 |
| ~image-title-to-caption.lua~ | 图片标题转为 Word 题注 |
| ~image-title-to-caption-add-number.lua~ | 同上 + 自动编号 |
| ~markdown-html-recognition.lua~ | 识别 Markdown 中的 HTML |
| ~preserve-font-color.lua~ | 保留字色 |

* 切换模板

~C-u C-c C-e~ 进入导出菜单，选择 Pandoc → docx。或在 ~SPC m e~
导出菜单中临时修改 ~org-pandoc-options-for-docx~ 中的 ~reference-doc~。

* 依赖

- Pandoc（系统安装）
- Lua（pandoc 内置，无需额外安装）
- Lua filter 所需的 lua 库（~pandoc/lua/~）
