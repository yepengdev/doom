#+TITLE: Spacious Padding

* 用途

在 Emacs 界面元素之间添加呼吸空间——fringe、header-line、mode-line 的间距，降低视觉拥挤感。

* 实现

三件事：
1. ~line-spacing~ = 3（行间额外垂直间距）
2. ~spacious-padding-mode~ 启用
3. 用 daemon-safe 模式确保终端中不启用

**Daemon-safe 模式**

#+BEGIN_SRC emacs-lisp
(defvar my/enable-spacious-padding--done nil)

(defun my/enable-spacious-padding--fn (&optional _frame)
  (when (and (display-graphic-p)
             (not my/enable-spacious-padding--done))
    (setq my/enable-spacious-padding--done t)
    (remove-hook 'doom-switch-frame-hook #'my/enable-spacious-padding--fn)
    (spacious-padding-mode 1)))

(use-package! spacious-padding
  :commands spacious-padding-mode
  :init
  (setq-default line-spacing 3)
  (add-hook 'doom-switch-frame-hook #'my/enable-spacious-padding--fn))
#+END_SRC

- ~display-graphic-p~ 确保只对 GUI frame 启用（终端 Emacs 的 padding 效果难以控制）
- ~my/enable-spacious-padding--done~ 是 once-only guard
- ~doom-switch-frame-hook~ 在第一个 GUI frame 出现时触发，之后自卸

* 效果

启用后：
- 左右 fringe 更宽
- mode-line 上下有 padding
- header-line 更舒适
- 行间间距 ~line-spacing~ = 3px

实际效果类似 Apple 生态（iTerm/Xcode）中普遍采用的宽松排版。
